# Contributing

Contributions are welcome.

- Open an issue: <https://github.com/ProjectCatMapper/CatMapPy/issues>
- Open a pull request: <https://github.com/ProjectCatMapper/CatMapPy/pulls>

## Development setup

```bash
python -m pip install -e .[dev,docs]
```

## Run tests

```bash
pytest -q
```

## Build docs locally

```bash
mkdocs serve
```
